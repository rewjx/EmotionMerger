# EmotionMerger

CG，立绘差分图片合成工具

不能保证实现的所有引擎都具备通用性,逆向能力不足是一方面，另一方面如果是二进制形式的CG表或坐标信息，有些引擎就难以做到通用。
部分实现为根据规律推测而来，更无法做到通用性。而且参考的游戏有限，难免有例外。

若发现不兼容，有能力的可以自己参考修改代码。

时间有限，随缘更新(弃坑)。
